Website Monitoring Sistem Fermentasi Nira Siwalan

Website ini dibuat untuk memonitoring hasil dari alat sistem fermentasi siwalan dan berisi data - data hasil fermentasi secara realtime. Mempunyai fitur auto refresh data pada bagian dashboard jadi tidak usah merefresh lagi, terdapat laporan data sebelumnya dalam bentuk tabel angka serta grafik dan mempunyai fitur print data dalam bentuk pdf.

Teknologi : Framework Code Igniter dan Bootstrap serta bahasa pemrograman HTML, CSS, PHP dan Javascript

Website Kopi Kenangan Senja

Website profil dari sebuah kedai kopi bernama kenangan senja yang digunakan untuk promosi kedai tersebut dan berisikan daftar menu yang ada pada kedai.

Teknologi : Bahasa Pemrograman HTML, CSS dan Javascript

Website Sarana Prasarana Peminjaman Alat
